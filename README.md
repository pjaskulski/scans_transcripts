# scans_transcripts

A simple scan and transcription viewer with the option of reading scans using the Gmini Pro 3 model and the ability to manually correct transcriptions. Additionally, a script for reading the entire series of scans by the Gemini Pro 3 model. The Gemini API key should be saved in the .env file as the GEMINI_API_KEY environment variable.
